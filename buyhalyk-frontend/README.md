# BuyHalyk Frontend

Современный маркетплейс для Казахстана, построенный на React + Vite + Tailwind CSS.

## Возможности

- 🏠 Главная страница с категориями и товарами
- 📈 Трендовые товары
- 💬 Система сообщений с продавцами
- 📜 История покупок
- ❤️ Избранные товары
- 🛒 Корзина покупок
- 👤 Личный кабинет
- ⚙️ Настройки (темная тема, язык)
- 🔐 Аутентификация (вход/регистрация)
- 🌍 Мультиязычность (RU, KK, EN)

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение откроется на `http://localhost:3000`

## Сборка

```bash
npm run build
```

## Структура проекта

```
src/
├── components/       # Переиспользуемые компоненты
├── pages/           # Страницы приложения
├── context/         # React Context для глобального состояния
├── utils/           # Утилиты и вспомогательные функции
├── App.jsx          # Главный компонент приложения
├── main.jsx         # Точка входа
└── index.css        # Глобальные стили
```

## Технологии

- React 18
- Vite
- Tailwind CSS
- Lucide React (иконки)
- React Context API (управление состоянием)
